Project Overview

• Developed a web-based platform designed to connect caregivers with individuals or families in need of caregiving services • Implemented job posting functionality, allowing individuals or organizations to create job listings with specific caregiving requirements • Built a caregiver registration module where caregivers can sign up, create profiles, and apply for jobs • Integrated an appointment scheduling system, allowing users to book and manage caregiving appointments • Designed and implemented the backend with PHP and MySQL, handling user authentication, job postings, and appointment data management

Technologies Used
